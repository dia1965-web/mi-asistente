# 📋 Mi Asistente Personal

Una aplicación web moderna y funcional para gestionar tus tareas personales de forma eficiente. Diseñada para ayudarte a organizar facturas, llamadas y trámites importantes.

## 🌐 Acceso a la App

**[Abre Mi Asistente Personal aquí](https://dia1965-web.github.io/mi-asistente/)**

## ✨ Características Principales

### 📝 Gestión de Tareas
- ✅ Crear tareas con título, tipo y fecha de vencimiento
- 📂 Clasificar tareas en 3 categorías:
  - 💵 **Pagos de Factura**
  - 📞 **Llamadas a Contactos**
  - 🏢 **Trámites de Oficina**
- 🔄 Marcar tareas como completadas
- 🗑️ Eliminar tareas individuales o todas a la vez

### 🔍 Búsqueda y Filtrado
- 🔎 Buscar tareas por título en tiempo real
- 📋 Filtrar por tipo de tarea
- 🎯 Encuentre rápidamente lo que necesita

### 📊 Estadísticas
- 📈 Panel de estadísticas con:
  - Total de tareas
  - Tareas completadas
  - Tareas pendientes
  - Tareas atrasadas
- 📥 **Exportar tareas a CSV** (descárgalas en Excel)

### 🌙 Tema Oscuro/Claro
- Alterna entre tema claro y oscuro
- Tu preferencia se guarda automáticamente
- Perfecto para usar en cualquier momento del día

### 🔔 Notificaciones
- 📣 Sonido y alerta cuando una tarea vence hoy
- ✉️ Mensajes visuales al agregar/eliminar tareas
- ⏰ Recordatorios automáticos

### 🎨 Indicadores de Urgencia
- **🔴 Rojo**: Tarea atrasada o vencida
- **🔴 Rojo brillante**: ¡Vence hoy!
- **🟡 Amarillo**: Vence en 3 días o menos
- **🔵 Azul**: Más de 3 días para vencer

### 💾 Almacenamiento Local
- Todas tus tareas se guardan en tu dispositivo
- **Privacidad garantizada**: Los datos nunca se envían a servidores
- Acceso instantáneo sin necesidad de internet (después del primer acceso)

## 🚀 Cómo Usar

### 1. **Agregar una Nueva Tarea**
   - Escribe lo que tienes que hacer en el campo "¿Qué tienes que hacer?"
   - Selecciona el tipo de tarea
   - Elige la fecha de vencimiento
   - Haz clic en "Agregar Tarea"

### 2. **Buscar Tareas**
   - Usa la barra de búsqueda para encontrar tareas por nombre
   - Usa el filtro para ver solo un tipo de tarea

### 3. **Marcar como Completada**
   - Haz clic en el botón "✔ Hecho" en la tarea
   - La tarea se marcará con tachado

### 4. **Eliminar una Tarea**
   - Haz clic en el botón "🗑️" para eliminar una tarea específica
   - Haz clic en "🗑️ Limpiar todo" para eliminar todas las tareas

### 5. **Ver Estadísticas**
   - Haz clic en el botón "📊" en la esquina superior derecha
   - Verás un panel con tus estadísticas

### 6. **Exportar a CSV**
   - Abre el panel de estadísticas
   - Haz clic en "📥 Exportar a CSV"
   - Se descargará un archivo que puedes abrir en Excel

### 7. **Cambiar Tema**
   - Haz clic en el botón "🌙" (o "☀️" en modo oscuro)
   - La app cambiará de tema automáticamente
